# Visualizador de Pendências SEV

Ferramenta pessoal para auxiliar na visualização das pendências de quilometragem dos relatórios MACUV do SDA. Desenvolvida por iniciativa própria para uso interno entre colegas.

## O que faz

- Lê os relatórios MACUV exportados do SDA (`.csv` com separador `;`)
- Exibe um painel consolidado com uso por viatura, pendências de quilometragem e todos os registros processados
- Permite cadastrar o nome dos motoristas a partir dos códigos presentes nos relatórios
- Tudo processado **localmente no navegador**, sem envio de dados a nenhum servidor

## Como usar

1. Acesse o SDA → Gestão de Veículos → Veículos → MACUV
2. Selecione o ano desejado e baixe o relatório de cada viatura
3. Abra cada `.xls` no Excel e salve como **CSV UTF-8** (separador `;`)
4. Mantenha a placa no nome do arquivo no formato `AAA-AAAA` (ex.: `SSP-1F53`)
5. Abra a ferramenta e selecione os arquivos gerados
