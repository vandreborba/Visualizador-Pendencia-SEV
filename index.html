Instruções

  1. Acesse o SDA → Gestão e Veículos → Veículos.
  2. Em MACUV, selecione o ano desejado.
  3. Baixe os relatórios de cada carro da agência.
  4. Abra cada arquivo `.xls` no Excel e salve como `.csv` com separador `;` e codificação UTF-8 (Arquivo → Salvar como → CSV UTF-8).
  5. Mantenha no nome do arquivo a placa do veículo no formato AAA-AAAA (ex.: SSP-1F53).

Selecione os arquivos dos relatórios

Este projeto é não oficial, criado apenas para facilitar a visualização das inconsistências dos relatórios. Nenhuma informação é enviada ou armazenada fora do seu computador. Uso livre e sem vínculo institucional.
